Huddl

Som-Base

Designed a React based web application called “Som-Base” in a team, allowing users to get a list of dishes based on their wine selection, or to get a list wines based on their dishe selection. We created this app mainly for waiters to suggest pairings for guests.
Technologies used: React.js, ReactStrap, Express.js, Heroku API
